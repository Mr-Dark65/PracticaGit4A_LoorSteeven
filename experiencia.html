Lenguajes de Programacion

  • Java

    • Java es versátil y ampliamente utilizado en aplicaciones empresariales, desarrollo móvil (Android) y sistemas embebidos. Es conocido por su portabilidad y su enfoque en la orientación a objetos.

  • Pyhton

    • Python es fácil de aprender y se utiliza en desarrollo web, análisis de datos, automatización de tareas y más. Su sintaxis clara y legible lo hace popular entre los programadores.

  • C++

    • C++ es eficiente y se utiliza en sistemas, juegos, aplicaciones de alto rendimiento y desarrollo de software de sistemas operativos.

  • Ruby

    • Ruby es conocido por su elegancia y productividad. Se utiliza en desarrollo web (con el framework Ruby on Rails) y scripting.

  • JavaScript

    • JavaScript es esencial para el desarrollo web. Se ejecuta en el navegador del cliente y permite crear interactividad en las páginas web.

Bases de Datos

A continuacion se mostrara un listado con las bases de datos con las que tengo experiencia:

  • MySQL

    • MySQL es una base de datos de código abierto ampliamente utilizada. Es rápida, escalable y compatible con una variedad de aplicaciones y lenguajes de programación. Se utiliza en sitios web, aplicaciones empresariales y más.

  • SQLite

    • SQLite es una base de datos ligera y sin servidor. Se utiliza en aplicaciones móviles, navegadores web y otros contextos donde se necesita una base de datos local.

  • Oracle DataBase

    • Oracle es una base de datos empresarial ampliamente utilizada. Ofrece alta disponibilidad, seguridad avanzada y soporte para transacciones complejas. Se utiliza en grandes sistemas empresariales.

  • PostgreSQL

    • PostgreSQL es otra base de datos de código abierto. Es conocida por su robustez, soporte para funciones avanzadas (como JSON y geolocalización) y su enfoque en la integridad de los datos.
